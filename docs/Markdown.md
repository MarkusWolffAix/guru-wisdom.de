# Markdown Syntax Übersicht

Diese Übersicht zeigt dir die wichtigsten Formatierungen für deine Weisheiten auf **guru-wisdom.de**.

---

## 1. Textauszeichnungen
Manchmal braucht eine Botschaft mehr Nachdruck:

* **Fett**: `**Deine Botschaft**` -> **Deine Botschaft**
* *Kursiv*: `*Sanfte Betonung*` -> *Sanfte Betonung*
* ~~Durchgestrichen~~: `~~Alter Pfad~~` -> ~~Alter Pfad~~
* `Code-Stil`: `` `Begriffe` `` -> `Begriffe` (ideal für Fachwörter)

---

## 2. Überschriften
Strukturiere deine Gedanken mit verschiedenen Ebenen:

# Überschrift Ebene 1 (Haupttitel)
## Überschrift Ebene 2 (Sektionen)
### Überschrift Ebene 3 (Unterpunkte)

---

## 3. Zitate (Wisdom-Style)
Das wichtigste Element für weise Sprüche:

> "Stille ist nicht leer, sie ist voller Antworten."
> — *Unbekannter Weiser*

---

## 4. Listen & Aufzählungen
Ideal für Übungen oder Prinzipien.

**Die drei Säulen:**
* Achtsamkeit
* Geduld
* Mitgefühl

**Schritt-für-Schritt Anleitung:**
1.  Einatmen
2.  Innehalten
3.  Ausatmen

---

## 5. Medien & Links
Binde Wissen aus anderen Quellen ein:

* **Link:** `[Besuche den Guru](https://guru-wisdom.de)`
* **Bild:** `![Naturaufnahme](/images/nature.jpg)`
* **YouTube (über unseren Shortcode):** `[youtube:dQw4w9WgXcQ]`
